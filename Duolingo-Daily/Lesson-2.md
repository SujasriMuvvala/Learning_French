LESSON-2
Common Words:
-----------------------
je – I

suis – am

un / une – a/an (masculine / feminine)

homme – man

femme – woman

garçon – boy

fille – girl

et – and
| French | English | Notes                    |
| ------ | ------- | ------------------------ |
| je     | I       | Subject pronoun          |
| tu     | you     | Informal "you"           |
| suis   | am      | From verb *être* (to be) |
| es     | are     | *tu es* = you are        |
| un     | a / an  | Masculine article        |
| une    | a / an  | Feminine article         |
| homme  | man     |                          |
| femme  | woman   |                          |
| garçon | boy     |                          |
| fille  | girl    |                          |
| et     | and     |                          |
